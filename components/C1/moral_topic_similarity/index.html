Moral-topic similarity

This component evaluates how desirable a Twitter post is to a user, based on the user’s preferences, in terms of topic and moral values, and on the topics and moral values detected from the post. While topics’ and moral information is provided by other components, such as Topic detection model and MFT values detection, the information usert∈Rpuser_t \in \mathbb{R}^pusert​∈Rp and userm∈Rquser_m \in \mathbb{R}^quserm​∈Rq from the user is assumed to be defined as a parameter of the system to be initialised to start the program. This setting is ideal to analyse a system through simulation analyses, as it is possible to fully control the user specifics.
