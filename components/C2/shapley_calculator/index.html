Shapley calculator

This component provides the functionality to compute the Shapley value of an individual norm nin_ini​ within a normative system NNN (where ni∈Nn_i \in Nni​∈N), with respect to a given value vvv. As explained in Alignment calculator, given a normative system NNN and a value vvv, it is possible to quantify the degree of alignment AlgnN,v\mathsf{Algn}_{N,v}AlgnN,v​ that the norms in NNN have with respect to value vvv. The Shapley value ϕi(v)\phi_i(v)ϕi​(v) of norm ni∈Nn_i \in Nni​∈N quantifies the contribution that the specific norm nin_ini​ makes to that alignment. It is defined as:

In the previous equation, the summation is taken over all the subsets of NNN where nin_ini​ is absent, N∖{ni}N \setminus \{n_i\}N∖{ni​}. In general, given a normative system NNN and a set of norm {nj,nk,...}⊆N\{n_j, n_k, ...\} \subseteq N{nj​,nk​,...}⊆N, the removal of {nj,nk,...}\{n_j, n_k, ...\}{nj​,nk​,...} from NNN is represented by a normative system N′=N∖{nj,nk,...}N' = N \setminus \{n_j, n_k, ...\}N′=N∖{nj​,nk​,...} where the values of the normative parameters tied to {nj,nk,...}\{n_j, n_k, ...\}{nj​,nk​,...} are substituted by their baseline quantities. These baseline normative parameters are drawn from a baseline normative system NbslN_{bsl}Nbsl​. This baseline normative system has the same norms as NNN, tied to the same normative parameters, but the values that these baseline normative parameters are selected in such a way that, when it is implemented on the model, it does not cause it to evolve in any way. In other words, the baseline normative system NbslN_{\text{bsl}}Nbsl​ causes transitions to not lead to any change in system state, s→Nbsls\mathbf{s} \xrightarrow{N_{\text{bsl}}} \mathbf{s}sNbsl​​s.

A Shapley Calculator is initialized similarly to the Alignment Calculator, by providing (i) a representation of the model or system being examined (i.e. the entity upon which norms apply to), and (ii) the semantics function fvf_vfv​ of the value of interest vvv whose alignment is computed by the component.
